# base

## Environment setup: MacOS

```
./setup.sh
```

Installs/configures:
- nvim
- zsh
- Binaries: fzf, ripgrep, tree

## Useful utilities

- Use python `csvkit`
  - `csvsql` for SQL queries on plaintext
  - `csvlook` for pretty-printing tables

```
python3 -m pip install pipx
python3 -m pipx install csvkit
```
